simetbox-openwrt-core - simetbox firmware core
----------------------------------------------

This package is responsible for all customization related to the SIMETBOX
device itself, but unrelated to the Measurement Engine and LMAP scheduler core.

For all web-interface-releated functionality, refer to package:
* simetbox-openwrt-luci

For the measurement engine and LMAP core, refer to the packges:
* simetbox-openwrt-simet-ma
* simetbox-openwrt-simet-lmapd
* simetbox-openwrt-base (legacy measurement engine)
